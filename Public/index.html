Platform Features

Everything a college needs to run fair elections

🏛️
Multi-College Isolation
Each college gets its own fully isolated election workspace. Complete independence with zero data overlap between institutions.
🗳️
Smart Voting Rules
Automatic enforcement of UG/PG voting levels, department-wise voting, and Lady Representative eligibility for female students.
🔒
Secure Authentication
Triple-layer login with email/register number, password, and unique security code prevents any unauthorized voting.
📊
Real-time Results
Live vote counts with beautiful visual breakdowns by seat, department, UG/PG level, and candidate profiles.
👥
CSV Bulk Import
Upload hundreds of students at once using CSV or Excel files. Auto-generate passwords and security codes instantly.
⏱️
Timed Elections
Schedule elections with precise start/end times. Voting opens and closes automatically. No manual intervention required.
Getting Started

How ELECTRA works

01
Register College
Admin signs up, creates an isolated election workspace for their institution.
02
Add Students
Upload student list manually or via CSV. Auto-credentials are generated for each student.
03
Candidates Apply
Eligible students submit their application. Admin reviews and approves candidates for each seat.
04
Vote & Publish
Students vote within the scheduled window. Admin publishes results for the entire college to view.
